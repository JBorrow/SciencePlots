Welcome to SciencePlots's documentation!
========================================

SciencePlots is a useful set of Matplotlib styles for scientific scientific
papers, presentations and theses.
Almost all styles use LaTeX as a background to render formulas 
Here you will find information about each style and the
`index of languages<languages>` supported
